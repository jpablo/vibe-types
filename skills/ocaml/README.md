# OCaml Type System Constraint Guide

> Status: **Planned** — contributions welcome.

This section will map OCaml's type system features (modules/functors, GADTs, polymorphic variants, first-class modules, etc.) to the constraints they can enforce at compile time.

---

## Structure

| Directory | Contents |
|-----------|----------|
| `catalog/` | One document per type system feature |
| `usecases/` | One document per compile-time constraint |

See the [main README](../README.md) for the full project overview and shared [appendix](../appendix/).
